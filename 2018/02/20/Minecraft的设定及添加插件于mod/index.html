Minecraft服务器的设定及添加插件与mod

2018-02-20 Minecraft

大家好，通过上一篇文章我相信大家一定搭建好了自己Minecraft服务器，可是可能存在一些问题，比如盗版用户无法登录，命令方块无法使用，服务器最大人数太小等等。那么在这篇文章里，你都会得到解决（大概吧）。

本章用到的软件与方法

修改文件，可以选用vim软件，直接就可以在xshell内修改文件vim指令(博客园）vim 官网
第二种方法就是直接使用FileZilla将所需文件弄到本地，然后在Windows下修改，然后上传到服务器即可。

服务器无法启动

服务器在启动时提示

  • You need to agree to the EULA in order to run the server. Go to eula.txt for more info.

那么就照他说的做，打开服务器内的Minecraft服务端文件夹，找到eula.txt文件，然后修改里面最后一行，改为eula=true，保存退出即可启动

盗版用户登录

既然玩Minecraft服务器了，最好还是买个正版2333（国际版，不是网易的）。当然，我想不可能每个人都有正版，那么没正版是不是进不去我们所开的服务器了呢？不是的，请按照接下来说的修改服务器配置文件即可使盗版玩家登录服务器。
在Minecraft服务端夹内找到server.properties文件，修改里面的

  • online-mode=true

这一行，改为

  • online-mode=false

保存，重启Minecraft服务器即可。

命令方块启用

修改server.properties文件里面的这一行

  • enable-command-block=false

改为

  • enable-command-block=true

保存重启Minecraft服务器即可。

服务器最大人数修改

照样修改server.properties文件，修改这一行

  • max-players=20

等号后的数字就是服务器最大人数。
修改后保存，重启服务器即可。

服务器标语修改

在Minecraft多人游戏选项中，你的服务器如果有信号，下部会显示一串标语
默认为A Minecraft Server，有人一定会说，这不够酷，我要改一下。
那我们就来改一下吧，修改server.properties文件（没错，还是它）内的这一行

  • motd=A Minecraft Server

等于号之后的就是标语，推荐输入English，中文有时有乱码。。。。
保存，重启服务器即可。

白名单开启

还是修改server.properties文件，找到

  • white-list=false

改为

  • white-list=true

保存重启服务器。然后在游戏中输入/whitelist add 玩家名给予该玩家白名单
或者在服务器文件white-list.txt里添加玩家名 多个玩家名要换行。

mod添加

直接把mod本体和依赖上传到服务器的mods文件，然后重启服务器即可，建议一个个加，不然会爆炸2333.

  • 补充：如果删除mod后无法启动服务器，请尝试输入/fml confirm修复地图即可。

插件添加

把插件本体和依赖上传到服务器plugins文件内，保存，重启服务器即可，还是建议一个个加，注意插件支持版本。。。。

  • 补充 插件指令使用存在权限，至于权限添加你找插件的网站会给的。

本章到此结束，希望大家游戏愉快。
